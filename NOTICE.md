# Source Attribution

This repository is a standalone analysis package and not a mirror of the original source repository.

The input dataset used here was derived from the public Bangkok rail station dataset referenced below:

- Source reference: https://www.kaggle.com/datasets/gusbell/thailand-public-train-data-bangkok-area

The source repository included a `CC0 1.0 Universal` license file at the time this analysis package was prepared.

This repository includes:

- the processed input file used by the analysis
- the analysis script
- the generated outputs
- documentation for this specific analysis package

If you publish this repository, keep this notice and the license information so the origin of the input dataset remains clear.
